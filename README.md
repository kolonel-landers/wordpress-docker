### 🇬🇧 English ###

# WordPress Docker Project

This project deploys a local WordPress site using **Docker** and **Docker Compose**. It works on both **Windows** and **Linux (WSL2)**.

## Features

* Latest WordPress version
* MySQL database
* Docker volumes for persistence (`wp-content`)
* Easy setup with Docker Compose
* Compatible with Windows and Linux

## Requirements

* Docker Desktop (Windows) or Docker & Docker Compose (Linux/WSL2)
* Git (optional for version control)

## Installation and Launch

1. Clone the project:

```bash
git clone https://github.com/kolonel-landers/wordpress-docker.git
cd wordpress-docker
```

2. Start the services:

```bash
docker-compose up -d
```

3. Access WordPress:
   Open `http://localhost:8081` in your browser.
